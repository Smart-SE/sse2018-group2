■実行方法
　各ファイルを同フォルダ内に格納
　python main.py で実行

■ファイル出力
　標準出力と同じ内容がファイルに出力される
　起動時に既存ファイルがある場合を想定し、データを空にする処理を追加。

■センサIDの取得コマンド
　ls -l /sys/bus/w1/devices/
　上記で表示される【28-xxxx】がセンサID

■csv出力形式
　"タイムスタンプ","センサID","温度"

■改善・検討事項
　センサIDの自動取得
　ネットワーク上にcsv出力

  温度の変化を取得。
